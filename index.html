Google Drive 設定 ☁️

Google Cloud Console で設定が必要です：
  1. Google Cloud Console を開く
  2. 新しいプロジェクトを作成
  3. 「APIとサービス」→「ライブラリ」で Google Drive API を有効化
  4. 「認証情報」→「OAuth 2.0 クライアントID」を作成（種類：ウェブアプリ）
  5. 承認済みの JavaScript 生成元にこのページの URL を追加
  6. クライアント ID を下に貼り付け
